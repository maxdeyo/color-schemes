# Color Scheme Generator

WIP color scheme generator made using React Native, Expo, react-native-paper and the npm color-scheme package.
Developed to run on Android.

## To Run
1. Have npm and expo installed.
2. Download repo.
3. Run ```npm i```
4. Run ```expo start```
